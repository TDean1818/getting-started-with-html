Hyper Text Markup Language can be stored on servers that are connected to the internet and retrieved by various web browsers, such as Chrome and Firefox, to be viewed by individual users.
There are inline and block elements. Block elements from an invisible box around text that can be altered to change that enclosed text.
Anchor tags and image tags can move you to another web page or show you an image from a source.
It is important to know that span and div tags can reference classes to change the styles of the various ones used.
